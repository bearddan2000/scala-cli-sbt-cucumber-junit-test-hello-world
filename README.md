# scala-cli-sbt-cucumber-junit-test-hello-world

## Description
A POC for sbt app using JUnit with
Cucumber plugin. Writes test results
to console and html.

## Tech stack
- scala
- sbt
  - junit
  - cucumber

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
